# 🎓 SISTEMA DE GESTÃO DE CURSOS

## PROJETO ACADÊMICO - TESTE E MANUTENÇÃO DE SISTEMAS

### 📋 RESUMO DO PROJETO
Sistema web completo desenvolvido em Django para gerenciamento de cursos, alunos e matrículas, com foco na aplicação de práticas de teste e manutenção de software.

### ✅ FUNCIONALIDADES IMPLEMENTADAS
1. **Autenticação Segura**
   - Login/Logout de usuários
   - Controle de sessões
   - Proteção de rotas

2. **Gestão de Cursos**
   - CRUD completo de cursos
   - Carga horária, datas, descrição
   - Status ativo/inativo

3. **Controle de Alunos**
   - Cadastro completo de alunos
   - CPF único, telefone, data de nascimento
   - Associação com usuários do sistema

4. **Matrículas**
   - Controle de matrículas em cursos
   - Status (ativa, concluída, cancelada)
   - Notas finais

5. **Dashboard e Relatórios**
   - Estatísticas do sistema
   - Visão geral para administradores
   - Interface responsiva

6. **Painel Administrativo**
   - Interface Django Admin customizada
   - Filtros, buscas e ações em massa
   - Controle de permissões

### 🧪 TESTES IMPLEMENTADOS
- **22 testes automatizados** (100% de sucesso)
- **5 categorias** de testes (Models, Views, URLs, Templates, Integração)
- **10 casos de teste manuais** documentados
- **Cobertura completa** das funcionalidades

### 📊 TECNOLOGIAS UTILIZADAS
- **Backend:** Django 5.1.3
- **Frontend:** HTML5, CSS3, JavaScript
- **Banco de Dados:** SQLite
- **Python:** 3.13
- **Testes:** Django Test Framework
