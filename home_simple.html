Bem-vindo ao Sistema de Gestão de Cursos

Uma plataforma completa desenvolvida em Django para gerenciamento acadêmico, criada para a disciplina de Teste e Manutenção de Sistemas.

✅ Sistema funcionando: Django 5.1.3 + Python 3.13
📚

🔧 Sistema de Administração

Plataforma completa para gerenciamento de todos os aspectos do sistema acadêmico:

  • ✅ CRUD Completo: Criar, Ler, Atualizar e Excluir registros
  • ✅ Controle de Usuários: Gerencie permissões e níveis de acesso
  • ✅ Relatórios: Visualize dados e estatísticas em tempo real
  • ✅ Validação de Dados: Sistema seguro com validação em todos os campos
